# Influencer Data

Place raw influencer CSV/spreadsheet data in this directory.

| Column | Type | Required |
|---|---|---|
| instagram_handle | @handle | ✅ |
| full_name | text | ❌ (can be scraped) |
| email | email | ❌ |
| phone | +91... | ❌ |
| follower_count | number | ❌ (can be scraped) |
| city | text | ✅ |
| content_niche | food/fashion/tech | ❌ |
